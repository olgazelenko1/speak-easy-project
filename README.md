# Speak Easy Project

This project is a React-based application using Vite as the build tool. It includes TypeScript for type safety and Firebase for backend services.

## Features

- React with TypeScript
- Firebase Authentication
- Modular CSS
- Vite for fast builds

## Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Build

To create a production build, run:

```bash
npm run build
```
